A simple Compiler Design project developed in C that demonstrates the core phases of a compiler, from source code analysis to error detection.

- Objective
To understand the fundamental working of a compiler by implementing its essential phases in C.


- Features
 Lexical Analysis (Token Generation)
 Syntax Analysis (Parsing)
 Semantic Analysis
 Symbol Table Management
 Error Detection & Reporting

 
- Technologies Used
Language: C
Concepts: DSA,Assembly language


- Build & Run
Compile
gcc *.c -o compiler
Run
./compiler


- Concepts Covered
Tokenization
Parsing
Semantic Checking
Symbol Table
Compiler Phases
Error Handling
