A simple Compiler Design project developed in C that demonstrates the core phases of a compiler, from source code analysis to error detection.</br>

- Objective
To understand the fundamental working of a compiler by implementing its essential phases in C.</br>


- Features</br>
 Lexical Analysis (Token Generation), 
 Syntax Analysis (Parsing), 
 Semantic Analysis, 
 Symbol Table Management, 
 Error Detection & Reporting</br>

 
- Technologies Used</br>
Language: C</br>
Concepts: DSA,Assembly language</br>


- Build & Run</br>
Compile: 
gcc *.c -o compiler</br>
Run: 
./compiler</br>


- Concepts Covered</br>
Tokenization, 
Parsing, 
Semantic Checking, 
Symbol Table, 
Compiler Phases, 
Error Handling</br>
